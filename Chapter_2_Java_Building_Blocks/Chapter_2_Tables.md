# Tables

## TABLE 2.3 Default initialization values by type

| Variable type                           | Default initialization value |
|-----------------------------------------|------------------------------|
| boolean                                 | false                        |
| byte , short , int , long               | 0                            |
| float , double                          | 0.0                          |
| char                                    | '\u0000'(NULL)               |
| All object references (everything else) | null                         |
